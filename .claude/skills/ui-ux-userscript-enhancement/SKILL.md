# UI UX Userscript Enhancement Skill

1. Preserve badge interaction contract (drag, toggles, settings, mode switch).
2. Keep keyboard accessibility and visible focus states.
3. Keep CSS changes scoped and maintain CSS variable pattern.
4. Validate viewport clamping and narrow-screen usability.
5. Run `bun run lint` and `bun run build`.
